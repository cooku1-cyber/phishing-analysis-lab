# phishing-analysis-lab

## Overview
This project documents a hands-on analysis of a cryptocurrency phishing kit.
The goal was to understand how attackers steal sensitive information and exfiltrate data.

## What is Phishing?
A phishing attack occurs when an attacker creates a fake or mimicked version of a legitimate website
to trick users into providing personal or sensitive information.

## Skills Demonstrated
- Phishing attack analysis
- Understanding HTML, JavaScript, and PHP roles
- Log analysis
- Identifying attacker infrastructure

## Key Takeaways
This lab helped me understand how phishing websites are built, how victim data is collected,
and how attackers receive stolen information.
